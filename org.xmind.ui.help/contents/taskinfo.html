In XMind Pro, you can add more detailed information to a topic in Taskinfo view. This can help you have good performance in TaskManage. You can add start date, end date, priority, and etc. easily. Here's how:

  1. Select a topic.
  2. OpenTaskInfo View.
  3. You can see and change following information in the view:
    • Name: this is content of selected topic;
    • Assign to: Type the people who are in charge of this task;
    • StartDate and EndDate: select the proper date for this task;
    • Progress: move the slide to ajust task progress;
    • Priority: select proper level for this task in the drop-down list;
    • Check Point: enable this check box will delete EndDate of this task.

Note: All topic's taskinfo will be shown in Gantt View too.
